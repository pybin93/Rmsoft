# Rmsoft - CodingTest


## 프로젝트 간단 소개
스프링부트를 이용하여 도서 관리 시스템을 구축했습니다.


## 사용된 기술

- 자바 버전 : Java 17
- 프레임 워크 : Spring Boot
- 데이터베이스 : MySql
- TEST TOOL : POSTMAN
- IDE : STS
- SERVER : AWS (신청은 했는데 실제 소스를 못올림)
